Evaluating Test Error of Cross-Validated Ridge

앞서 적절한 \(\lambda\)를 찾기 위해 cross-validation을 활용하는 방법을 살펴보았음.
타당한 “test” error를 구하기 위해서는 처음에 training set만을 이용해 \(\lambda\)를 찾은 후,
test set을 이용해 error를 구해야 함.
