Expected value

\(X: \{a, b, a, a, b, c\}\)
The mean of \(X=\displaystyle \frac{a + b + a + a + b + c}{6} = \frac{3*a}{6} + \frac{2*b}{6} + \frac{1*c}{6}=a*\frac{3}{6} + b*\frac{2}{6} + c*\frac{1}{6}\)
\(~~~~~~~~~~~~~~~~~~~~~~~~~=a*p(X=a) + b*p(X=b) + c*(X=c)\): weighted average

Expectation: \(E(X)=\displaystyle \sum x_i p(X=x_i)= \sum xp(x)\)

мЧ∞мЖНк∞ТмЭілЭЉл©і, \(E(X)=\displaystyle \sum \Delta x * p(\Delta x) = \int x f(x) dx\), where \(f(x)\) is the probability density function
\(p(\Delta x) = f(x)\Delta x\)

The BiasвАУVariance Trade-off

мЬДмЭШ лЕЉмЭШлКФ нКєм†Х лН∞мЭінД∞мЕЛмЧР мЭШм°інХ† мИШл∞ЦмЧР мЧЖлКФлН∞, (м∞Єк≥†: BayesianмЭШ к≤љмЪ∞ лЛ§л•Є м†СкЈЉ)
лН∞мЭінД∞мЕЛлІИлЛ§ лЛ§л•Є \(f\)л•Љ мЦїк≤М лРЬлЛ§лКФ м†РмЭД к∞РмХИнЦИмЭД лХМ, \(f\)мЭШ л≥АлПЩмД±мЭД мВінОіл≥іл©і,

\(L_2\) lossмЭШ к≤љмЪ∞,

\(h(x) := E(t|X=x)\): optimal prediction of Y at any point x

\(E(L_2) = \displaystyle \iint \left( f(x) - t \right)^2 p(x, t) dxdt = \iint \left( f(x) - h(x) + h(x) - t \right)^2 p(x, t) dxdt\) \(~~~~~~~~~~~ = \displaystyle \int \left( f(x) - h(x) \right)^2 p(x) dx + \iint \left( h(x) - t) \right)^2p(x, t) dxdt\)

  • лСР л≤ИмІЄ нХ≠: optimal prediction \(h(x)\)к∞А true valueмЩА мЦЉлІИлВШ лЦ®мЦім†Є мЮИлКФк∞А?(лґДмВ∞) - irreducible error
  • м≤Ђ л≤ИмІЄ нХ≠: \(h(x)\)мЩАмЭШ м∞®мЭіл•Љ мµЬмЖМнЩФнХШлПДл°Э \(f(x)\)л•Љ мД†нГЭнХШлКФлН∞ нХШлКФлН∞, мЬ†нХЬнХЬ лН∞мЭінД∞мЕЛмЧРмДЬлКФ кЈЄ к∞ДкЈємЭі м°імЮђ
    • мЭім†Ь, лґДнПђ \(p(t, x)\)л°ЬлґАнД∞ мИШлІОмЭА лН∞мЭінД∞мЕЛлУ§ \(D_i\)л•Љ мЦїмЧИлЛ§к≥† к∞Ам†ХнЦИмЭД лХМ,
    • нКєм†Х \(x_0\)мЧР лМАнХімДЬ

\(E_D[\left( f(x_0; D_i) - h(x_0)\right)^2]=[E_D\left( f(x_0; D_i)\right) - h(x_0)]^2 + E_D[\left\{f(x_0;D_i) - E_D\left(f(x_0;D_i)\right)\right\}^2]\)
\(~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ = bias^2 + variance\)

Bias: мµЬм†БмЭШ мШИмЄ°к∞Т(\(h\))мЧР лєДнХі, (мЧђлЯђ лН∞мЭінД∞л°ЬлґАнД∞ мЦїмЭА) вАЬнПЙкЈ†м†БмЭЄ мШИмЄ°к∞ТвАЭмЭі мЦЉлІИлВШ м∞®мЭік∞А лВШлКФк∞А?

  • м¶Й, мµЬм†БмЭШ мШИмЄ°л≥ілЛ§ нПЙкЈ†м†БмЬЉл°Ь мЦЉлІИлВШ нЛАл¶∞ мШИмЄ°мЭД нХШлКФк∞А?
  • the error that is introduced by approximating a real-life problem, which may be extremely complicated, by a much simpler model.

Variance: к∞Б лН∞мЭінД∞л°ЬлґАнД∞ мЦїмЭА мШИмЄ°к∞ТмЭі (мЧђлЯђ лН∞мЭінД∞лґАнД∞ мЦїмЭА) вАЬнПЙкЈ†м†БмЭЄ мШИмЄ°к∞ТвАЭмЧРмДЬ мЦЉлІИлВШ лЦ®мЦім†Є мЮИлКФк∞А?(лґДмВ∞)

  • м¶Й, лН∞мЭінД∞мЕЛмЧР лФ∞лЭЉ \(f(x_0;D_i)\)к∞А мЦЉлІИлВШ лѓЉк∞РнХШк≤М л≥АнХШлКФмІА мЄ°м†Х

л™®лУ† \(X\)мЧР лМАнХЬ нПЙкЈ†к∞ТмЬЉл°Ь мЭінХінХШл©і,
Expected loss \(\displaystyle E(L_2) = \left( \int (bias)^2 + \int variance \right) + noise\)

BiasмЩА varianceк∞ДмЧРлКФ trade-offк∞А м°імЮђнХ®

  • FlexibleнХЬ л™®лНЄмЭШ к≤љмЪ∞ лН∞мЭінД∞мЧР лНФ мЮШ м†БнХ©нХШмЧђ, varianceк∞А лЖТмХДмІР. м¶Й лН∞мЭінД∞лІИлЛ§ лДИлђі лЛ§л•Є мШИмЄ°мЭД нХШк≤М лР®
    • лН∞мЭінД∞мЧР overfittingмЭі лРЬлЛ§к≥† лІРнХ† мИШ мЮИмЭМ.
    • нХЬнОЄ, нПЙкЈ†к∞ТмЭі мЛ§м†Ьк∞ТмЧР к∞АкєМмЫМм†Є biasк∞А м§ДмЦілУ¶
    • лЛ®, NмЭі м¶Эк∞АнХШл©і(лН∞мЭінД∞ мВђмЭім¶Ик∞А мї§мІАл©і), flexibleнХЬ л™®лНЄмЭШ varianceк∞А м§ДмЦілУ¶.
  • RigidнХЬ л™®лНЄмЭШ к≤љмЪ∞ лН∞мЭінД∞мЧР лНЬ м†БнХ©нХШмІАлІМ, biasк∞А лЖТмХДмІР. м¶Й, нПЙкЈ†м†БмЬЉл°Ь лНФ нЛАл¶∞ мШИмЄ°мЭД нХШк≤М лР®
    • нХЬнОЄ, лН∞мЭінД∞мЧР лНЬ лѓЉк∞РнХШмЧђ varianceк∞А м§ДмЦілУ¶
    • мЮСмЭА лН∞мЭінД∞мЕЛмЧРмДЬ лНФ мЬ†л¶ђ
  • мЭі лСШмЭШ м†Бм†ИнХЬ кЈ†нШХмЭД к∞ЦлКФ л™®лНЄмЭі мµЬм†БмЭШ мШИмЄ° л™®лНЄмЮД

л™®лНЄмЭШ нПЙк∞АлКФ мШИмЄ°нХ®мИШмЭШ мШ§м∞®л°Ь мЭЄнХЬ мЖРмЛ§(loss)мЭі мµЬмЖМмЭЄ к≤ГмЬЉл°Ь кЄ∞м§АмЭД лСШ мИШ мЮИмЬЉлВШ,

k-fold CV estimate: \(\displaystyle CV_{(k)} = \frac{1}{k} \sum_{i=1}^k MSE_i\)

лґДл•ШмЭШ к≤љмЪ∞, misclassification error rate: \(\displaystyle CV_{(k)} = \frac{1}{k} \sum_{i=1}^k \frac{1}{n_i} \sum_{j=1}^{n_i} I(y_j \neq \hat y_j)\)
